Sistem Antrian Customer Menggunakan Stack Array

Program ini merupakan aplikasi sederhana berbasis Python yang digunakan untuk mengelola antrian customer pada sebuah toko bernama Toko Iskandar Jaya. Program memungkinkan pengguna untuk menambahkan customer ke dalam antrian, memanggil customer, melihat customer terdepan, serta menampilkan seluruh isi antrian. Sistem dijalankan melalui menu interaktif sehingga pengguna dapat memilih operasi yang diinginkan.

Pada bagian awal program dibuat class bernama StackArray yang digunakan untuk mengelola seluruh proses stack. Class ini menjadi struktur utama dalam penyimpanan data customer. Di dalam konstruktor (__init__), program menginisialisasi kapasitas maksimum stack menggunakan variabel MAX. Nilai default kapasitas adalah 1000 data. Selanjutnya dibuat list bernama st dengan ukuran sesuai kapasitas maksimum. Seluruh isi list awalnya bernilai None sebagai penanda bahwa data masih kosong. Program juga membuat variabel top_idx dengan nilai awal -1. Variabel ini berfungsi sebagai penunjuk posisi data paling atas pada stack. Nilai -1 menandakan bahwa stack masih kosong dan belum memiliki data. 
Fungsi is_empty digunakan untuk memeriksa apakah stack kosong atau tidak. Program akan mengecek apakah nilai top_idx sama dengan -1. Jika benar, maka fungsi mengembalikan nilai True yang berarti tidak ada data di dalam stack. Jika tidak, maka fungsi mengembalikan nilai False.
Fungsi is_full digunakan untuk memeriksa apakah stack sudah penuh. Program membandingkan nilai top_idx dengan kapasitas maksimum dikurangi satu (MAX - 1). Jika kedua nilai sama, berarti seluruh kapasitas array telah terisi dan stack tidak dapat menerima data baru lagi. Fungsi ini digunakan untuk mencegah kondisi overflow.
Fungsi push digunakan untuk menambahkan customer ke dalam stack. Program pertama kali memeriksa apakah stack penuh dengan memanggil fungsi is_full. Jika penuh, maka program menampilkan pesan “Antrian Penuh” dan proses dihentikan menggunakan return. Jika stack masih memiliki ruang kosong, maka nilai top_idx ditambah satu sebagai penanda posisi data baru. Setelah itu, nama customer disimpan pada indeks tersebut di dalam array st. Setelah data berhasil dimasukkan, program menampilkan pesan bahwa customer telah masuk ke dalam antrian. Karena menggunakan konsep stack, data yang terakhir dimasukkan akan berada di posisi paling atas.
Fungsi pop digunakan untuk menghapus atau memanggil customer dari stack. Program terlebih dahulu memeriksa apakah stack kosong menggunakan fungsi is_empty. Jika kosong, program menampilkan pesan “Antrian Kosong” lalu menghentikan proses. Jika stack tidak kosong, program menampilkan nama customer yang berada di posisi paling atas. Setelah itu, nilai top_idx dikurangi satu sehingga data paling atas dianggap keluar dari stack. Karena stack menggunakan prinsip LIFO, customer yang terakhir masuk akan dipanggil terlebih dahulu.
Fungsi peek digunakan untuk melihat data customer yang berada di posisi paling atas tanpa menghapus data tersebut. Program akan mengecek apakah stack kosong. Jika kosong, maka ditampilkan pesan bahwa antrian kosong. Jika masih ada data, program menampilkan customer yang berada pada indeks top_idx. Data tetap berada di dalam stack dan tidak dihapus.
Fungsi display digunakan untuk menampilkan seluruh isi stack. Program pertama kali memeriksa apakah stack kosong. Jika kosong, maka ditampilkan pesan “Antrian Kosong”. Jika stack memiliki data, program menampilkan seluruh customer mulai dari posisi paling atas hingga posisi paling bawah menggunakan perulangan menurun. Perulangan dimulai dari top_idx sampai indeks 0. Hal ini dilakukan agar urutan tampilan sesuai konsep stack, yaitu data terbaru tampil lebih dahulu.
