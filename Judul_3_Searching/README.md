Sistem Pencarian Nama Pasien Menggunakan Sequential Search

Program ini merupakan aplikasi sederhana berbasis Python yang digunakan untuk melakukan pencarian nama pasien pada sebuah daftar data pasien. Pengguna diminta memasukkan beberapa nama pasien, kemudian program akan mencari apakah nama yang diinput tersedia di dalam daftar tersebut.

Pada bagian awal program dibuat fungsi bernama sequential_search yang berfungsi untuk melakukan proses pencarian data pasien. Fungsi ini menerima dua parameter, yaitu list nama pasien dan nama target yang ingin dicari. Di dalam fungsi tersebut dibuat variabel counter dengan nilai awal 0 yang digunakan untuk menghitung jumlah data yang cocok dengan target pencarian. Program kemudian melakukan perulangan untuk membaca setiap nama pasien di dalam list. Setiap data dibandingkan dengan target menggunakan method lower() agar pencarian tidak membedakan huruf besar dan huruf kecil. Dengan demikian, input seperti BENI, Beni, atau beni tetap dianggap sama. Jika data yang diperiksa sesuai dengan target pencarian, maka nilai counter akan bertambah satu. Setelah seluruh data selesai diperiksa, fungsi akan mengembalikan nilai counter sebagai hasil pencarian.
Bagian berikutnya adalah fungsi main yang menjadi pusat jalannya program. Program pertama kali meminta pengguna memasukkan daftar nama pasien. Input yang dimasukkan dipisahkan menggunakan fungsi split() sehingga setiap nama otomatis tersimpan sebagai elemen di dalam list. Setelah itu, program menampilkan kembali daftar pasien yang telah dimasukkan agar pengguna dapat melihat data yang tersimpan. Selanjutnya pengguna diminta memasukkan nama pasien yang ingin dicari.
Setelah nama target dimasukkan, program memanggil fungsi sequential_search untuk melakukan proses pencarian. Hasil pencarian disimpan ke dalam variabel counter. Program kemudian melakukan pengecekan terhadap nilai tersebut. Jika nilai counter lebih besar dari nol, maka program menyimpulkan bahwa nama pasien ditemukan di dalam daftar dan menampilkan pesan bahwa pasien ditemukan. Namun jika nilai counter sama dengan nol, maka program menampilkan pesan bahwa pasien tidak ditemukan.
Sebagai contoh, jika pengguna memasukkan daftar nama pasien seperti Ali Vian Beni, kemudian mencari nama Beni, maka program akan memeriksa setiap data secara berurutan hingga menemukan nama yang sesuai. Setelah ditemukan, program akan menampilkan pesan bahwa pasien atas nama Budi ditemukan. Sebaliknya, jika pengguna mencari nama yang tidak terdapat di dalam daftar, misalnya “Rina”, maka program tetap memeriksa seluruh data hingga akhir dan akhirnya menampilkan pesan bahwa pasien tidak ditemukan.

Secara keseluruhan, program ini berhasil menerapkan algoritma Sequential Search dalam proses pencarian data pasien. Program dapat menerima input daftar pasien, melakukan pencarian data secara berurutan, serta menampilkan hasil pencarian dengan benar. Selain itu, penggunaan method lower() membuat proses pencarian menjadi lebih fleksibel karena tidak sensitif terhadap perbedaan huruf besar dan kecil. Program ini menunjukkan bagaimana konsep dasar algoritma pencarian dapat diterapkan dalam studi kasus sederhana yang mudah dipahami.
